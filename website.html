Ultrasonic Distance Measurement and LED Indication System using Arduino

Abstract

This project demonstrates an Arduino-based ultrasonic distance measurement system that uses an HC-SR04 sensor to measure the distance to an object and activates an LED when it crosses a set threshold. It highlights automation and real-time sensing in low-cost embedded systems.

System Components

  • Arduino UNO Microcontroller
  • HC-SR04 Ultrasonic Sensor
  • LED Indicator
  • Potentiometer (for Threshold Adjustment)
  • Breadboard & Jumper Wires
